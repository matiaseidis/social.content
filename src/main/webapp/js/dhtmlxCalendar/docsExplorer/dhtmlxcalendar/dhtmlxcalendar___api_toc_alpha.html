dhtmlxCalendar API Methods

dhtmlxDblCalendarObject Object

  • attachEvent attaches a event handler to calendar
  • detachEvent removes an event handler
  • draw draws calendar
  • hide hides calendar
  • isVisible gets visibility state
  • loadUserLanguage sets new language interface for calendar
  • minimize minimizes calendar
  • setDate selects date in calendar
  • setDateFormat sets date format for input/output
  • setHeader sets header in calendar
  • setHeaderText sets header title
  • setHolidays sets holidays dates
  • setOnClickHandler sets onClick event handler
  • setParent sets parent node for calendar
  • setSensitive sets sensitive range for calendar
  • setYearsRange sets years range
  • show shows calendar
